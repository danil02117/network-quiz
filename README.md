# 📡 Сетевые технологии - Мобильное приложение для подготовки к тесту

## 📱 Как установить на телефон (чтобы работало БЕЗ компьютера)

### ⚡ Способ 1: Netlify Drop (рекомендуется — 2 минуты)

1. Откройте сайт **https://app.netlify.com/drop**
2. Перетащите **всю папку** `e:\test` в окно браузера
3. Через несколько секунд получите ссылку вида `https://random-name.netlify.app`
4. Откройте ссылку на телефоне
5. **Добавьте на главный экран:**
   - **Android (Chrome):** меню (⋮) → "Добавить на главный экран"
   - **iOS (Safari):** кнопка "Поделиться" → "На экран «Домой»"
6. Готово! Приложение работает независимо от компьютера

### 🐙 Способ 2: GitHub Pages (навсегда, бесплатно)

1. Зарегистрируйтесь на **https://github.com**
2. Создайте новый репозиторий (назовите, например, `network-quiz`)
3. Загрузите все файлы через кнопку **"Upload files"**
4. Зайдите в **Settings → Pages**
5. В разделе **Source** выберите ветку `main` → **Save**
6. Через 1-2 минуты сайт доступен по ссылке `https://ваш-ник.github.io/network-quiz/`
7. Откройте на телефоне и добавьте на главный экран (см. выше)

### 📦 Другие варианты хостинга

- **Vercel** (https://vercel.com)
- **Cloudflare Pages** (https://pages.cloudflare.com)
- **Firebase Hosting** (https://firebase.google.com)

## 📲 Как установить как приложение

### Android (Chrome):
1. Откройте сайт в Chrome
2. Нажмите меню (⋮) → **"Добавить на главный экран"**
3. Приложение появится на рабочем столе

### iOS (Safari):
1. Откройте сайт в Safari
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой)
3. Выберите **"На экран «Домой»"**
4. Приложение появится на рабочем столе

## 🎮 Как пользоваться

1. **Выберите тему** из выпадающего списка
2. **Прочитайте вопрос** и выберите вариант ответа
3. Нажмите **"Подтвердить"**
4. Увидите правильный ответ и объяснение
5. После всех вопросов получите результат с процентом правильных ответов
